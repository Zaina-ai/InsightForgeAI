# InsightForge AI

AI-powered customer feedback intelligence platform.

## Features

* Upload CSV customer feedback
* Sentiment Analysis
* Theme Detection
* Priority Recommendations
* Interactive Dashboard

## Tech Stack

Frontend:

* Next.js
* React
* Tailwind CSS

Backend:

* FastAPI
* Python

Machine Learning:

* Pandas
* TextBlob
